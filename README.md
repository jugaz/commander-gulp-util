## Installation

```bash
$ npm install commander-gulp-util
```


#### Command to Compile

```bash
$ commander-gulp-util util 'entry' --t 'ouput' 
```

#### Example

```bash
 "scripts": {
    "util": "commander-gulp-util util \"frontend/src/fonts\" --ut  \"docs/\"",
  }
```
